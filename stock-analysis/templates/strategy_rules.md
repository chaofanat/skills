# 股票技术分析 - 分阶段策略规则

本文件包含五步分析法中的详细判定标准和策略模板，与 SKILL.md 的核心框架配合使用。

**设计原则**：从"硬性规则"转向"评分机制"，从"二元判断"转向"概率判断"，从"固定阈值"转向"动态适配"。

---

## 一、趋势阶段细分规则

当第二步判定为"趋势加速市"时，必须执行此细分判定。

### 趋势加速三阶段评分标准

| 阶段 | 评分项 | 得分标准 | 阈值 |
|------|--------|---------|------|
| **加速初期** | 累计涨幅 | <20%: +2分, 20-30%: +1分, >30%: 0分 | ≥4分 |
| | 乖离率 | 10日<12%: +2分, 12-15%: +1分, >15%: 0分 | |
| | 量能 | 放量>20%: +2分, 持平: +1分, 缩量: 0分 | |
| | 平台突破 | 刚突破: +2分, 已突破一段: +1分 | |
| **加速中期** | 累计涨幅 | 20-30%: +2分, 30-40%: +1分 | ≥4分 |
| | 乖离率 | 10日12-18%: +2分, 其他: +1分 | |
| | 量能 | 持续: +2分, 递减: +1分 | |
| | 均线 | 完美发散: +2分, 部分发散: +1分 | |
| **加速末期** | 累计涨幅 | >35%: +3分, 30-35%: +1分 | ≥6分 |
| | 乖离率 | 10日>18%: +3分, 15-18%: +1分 | |
| | 超买指标 | RSI>75或KDJ>90: +2分 | |
| | 量价配合 | 缩量或背离: +2分 | |

### 判定逻辑

```
1. 计算各阶段得分
2. 得分最高的阶段为当前阶段
3. 如最高分<4分：处于震荡市，非趋势加速
4. 如加速末期≥6分：进入高风险区，策略需保守
```

### 动态阈值调整

| 市场环境 | 涨幅阈值调整 | BIAS阈值调整 | 说明 |
|---------|-------------|-------------|------|
| 强势牛市 | +10% | +3% | 龙头股可涨幅更大 |
| 弱势熊市 | -10% | -3% | 更早判定风险 |
| 震荡市 | 不变 | 不变 | 标准阈值 |
| 个股属性 | | | |
| - 高波成长股 | +15% | +5% | 波动大是常态 |
| - 稳健蓝筹股 | -5% | -2% | 波动小应更敏感 |

## 二、信号权重评估规则

在第一步信号收集后，必须执行加权评估。

### 综合评分机制

```
总得分 = 趋势强度(30%) + 量价配合(25%) + 资金流向(20%) + 乖离程度(15%) + 动量状态(10%)
```

#### 趋势强度（30分）

| 指标 | 评分标准 |
|------|---------|
| ADX值 | >40: 10分, 30-40: 8分, 25-30: 5分, <25: 2分 |
| MA排列 | 完美多头: 10分, 部分多头: 6分, 粘合: 3分 |
| MACD | 强多头(HIST>0.5): 10分, 弱多头: 6分, 其他: 0分 |

#### 量价配合（25分）

| 指标 | 评分标准 |
|------|---------|
| 量价关系 | 放量上涨: 10分, 平量上涨: 6分, 缩量上涨: 2分 |
| OBV趋势 | 强上升: 10分, 弱上升: 6分, 走平/下降: 0分 |
| 布林带 | 突破上轨: 5分, 接近上轨: 3分, 中轨下方: 0分 |

#### 资金流向（20分）

| 指标 | 评分标准 |
|------|---------|
| 主力净流入 | >+5000万: 10分, +1000-5000万: 6分, <0: 0分 |
| 5日累计 | >+1亿: 10分, +5000-1亿: 6分, <0: 0分 |

#### 乖离程度（15分）

| 指标 | 评分标准 |
|------|---------|
| 10日BIAS | <8%: 5分, 8-15%: 3分, 15-20%: 1分, >20%: -3分 |
| 20日BIAS | <15%: 5分, 15-22%: 2分, >22%: -3分 |
| 距前低涨幅 | <30%: 5分, 30-50%: 2分, >50%: -2分 |

#### 动量状态（10分）

| 指标 | 评分标准 |
|------|---------|
| RSI(14) | 50-70: 10分, 70-80: 3分, >80: -5分, <50: 0分 |
| KDJ | K>D且未超买: 5分, 超买: 0分, 超买且钝化: -3分 |

### 特殊调整项

| 调整项 | 扣分 | 说明 |
|--------|------|------|
| 明显量价背离 | -15分 | 价格新高+量能递减 |
| 大单出逃 | -10分 | 大单流出>5000万 |
| 大单环境恶化 | -20分 | 大盘当日跌幅>3% |
| 涨停板 | +5分 | 涨停缩量是正常现象 |
| 高控盘特征 | +3分 | 筹码锁定可缩量上涨 |

### 综合评级标准

| 总分 | 评级 | 含义 | 建议仓位 |
|------|------|------|---------|
| ≥90 | ⭐⭐⭐⭐⭐ 明确看多 | 五维共振向上，上涨概率高 | 50-70% |
| 75-89 | ⭐⭐⭐⭐ 偏多 | 五维偏多，机会大于风险 | 30-50% |
| 60-74 | ⭐⭐⭐ 中性 | 五维分歧，观望为主 | 10-30% |
| 45-59 | ⭐⭐ 偏空 | 五维偏空，风险大于机会 | 0-10% |
| <45 | ⭐ 明确看空 | 五维共振向下，下跌概率高 | 0% |

**注意**：评分是概率判断，非绝对预测。90分代表"高概率上涨"，非"一定上涨"。

## 三、动态信号解读规则（含组合信号）

### 组合信号解读表（概率化表达）

| 组合信号 | 强势市场 | 震荡市场 | 弱势市场 |
|---------|---------|---------|---------|
| **放量上涨** | 80%向上 | 65%向上 | 50%向上 |
| **缩量上涨+资金流入** | 70%向上 | 50%中性 | 40%向下 |
| **缩量上涨+资金流出** | 40%向上(诱多) | 30%向下 | 20%向下 |
| **放量下跌** | 40%向下(洗盘) | 60%向下 | 70%向下 |
| **缩量下跌** | 50%中性(洗盘) | 50%中性 | 60%向下 |

### 例外情况识别

| 情况 | 解读调整 |
|------|---------|
| **涨停板缩量** | 正常现象，不视为量价背离 |
| **高控盘股缩量上涨** | 可能是筹码锁定，不视为风险 |
| **市场整体缩量** | 非个股原因，降低负面解读 |
| **重大利好消息** | 情绪驱动，量能要求降低 |
| **除权除息日** | 价格失真，需复权分析 |

### 信号强度分层

| 强度 | 特征 | 操作含义 |
|------|------|---------|
| 🔴 极强信号 | 多维度共振+无背离+放量 | 可积极介入 |
| 🟠 强信号 | 多数维度配合+轻微背离 | 谨慎介入 |
| 🟡 中性信号 | 维度分歧 | 观望 |
| 🟢 弱信号 | 少数维度配合+明显背离 | 规避 |

## 四、背离检测规则（评分化）

在第四步共振验证时，必须执行背离评分。

### 背离类型与评分

| 背离类型 | 判定标准 | 扣分 | 说明 |
|---------|---------|------|------|
| **量价背离** | 价格新高 + 量能环比-15%以上 | -8分 | 动能衰竭 |
| **量价背离** | 价格新高 + 量比<0.8 | -5分 | 轻微背离 |
| **量能背离** | 价格上涨 + 大单净流出>5000万 | -10分 | 资金出逃 |
| **量能背离** | 价格上涨 + 大单净流出2000-5000万 | -5分 | 资金分歧 |
| **动量背离** | 价格新高 + RSI/KDJ顶背离 | -10分 | 确认背离 |
| **动量背离** | 价格新高 + RSI/KDJ高位钝化 | -5分 | 潜在背离 |

### 背离豁免条件

出现以下情况时，背离扣分减半或豁免：

| 情况 | 处理 | 说明 |
|------|------|------|
| 涨停板 | 豁免量价背离 | 涨停缩量正常 |
| 高控盘股 | 扣分减半 | 机构锁仓 |
| 市场整体缩量 | 扣分减半 | 非个股原因 |
| 突破重要阻力 | 扣分减半 | 解套盘压制 |
| 消息驱动 | 扣分减半 | 情绪主导 |

### 背离综合判定

```
背离总分 = 各背离项扣分之和 - 豁免减分

背离总分 < 5分：轻微背离，可忽略
背离总分 5-15分：中等背离，需警惕
背离总分 > 15分：严重背离，降低评级
```

## 五、分阶段操作策略模板

### 交易周期分层

| 周期类型 | 持有期 | 风险容忍 | 策略特点 |
|---------|--------|---------|---------|
| 超短线 | <3天 | 高 | 严格止损，可追涨 |
| 短线 | 3-10天 | 中 | 等待确认，控制仓位 |
| 中长线 | >10天 | 低 | 趋势跟踪，分批建仓 |

---

### 震荡市策略（评分60-74分）

| 类型 | 介入条件 | 止损位 | 目标位 | 仓位 |
|------|---------|--------|--------|------|
| 稳健型 | 箱体下沿/支撑位确认 | 跌破箱体下沿-3% | 箱体上沿-5% | 30% |
| 短线型 | 突破箱体上沿+放量 | 跌破突破位 | +10-15% | 20% |
| 超短线 | 突破当日追涨 | 当日收盘-3% | +5-8% | 10% |

---

### 加速初期策略（评分≥75分，初期判定≥4分）

| 类型 | 介入条件 | 止损位 | 目标位 | 仓位 |
|------|---------|--------|--------|------|
| 稳健型 | 回踩MA10/前平台+缩量 | 跌破MA20 | +20-30% | 40-50% |
| 短线型 | 突破确认后+放量 | 跌破MA5 | +25%+ | 30-40% |
| 超短线 | 突破当日+强势涨停 | 次日开盘-2% | +10-15% | 20% |

---

### 加速中期策略（评分70-84分，中期判定≥4分）

| 类型 | 介入条件 | 止损位 | 目标位 | 仓位 |
|------|---------|--------|--------|------|
| 稳健型 | 浅调5-8%+缩量+踩支撑 | 跌破MA5 | +10-15% | 20-30% |
| 短线型 | 小仓位顺势+严格止损 | 严格破MA5 | +15-20% | 20% |
| 超短线 | 强势延续追涨 | 当日破前低 | +5-8% | 10% |

---

### 加速末期策略（末期判定≥6分）

| 类型 | 介入条件 | 止损位 | 目标位 | 仓位 |
|------|---------|--------|--------|------|
| 稳健型 | **观望为主，不新开仓** | - | - | 0% |
| 短线型 | **极低仓位参与反弹** | 严格止损 | +3-5% | 5-10% |
| 超短线 | **涨停确认可小仓位** | 次日破位即走 | +5% | ≤10% |
| 持筹者 | - | 破MA5/MA10离场 | - | 逐步减仓至≤30% |

**说明**：
- 加速末期并非"完全禁止"，而是"极低概率"
- 超短线交易者可在强势延续时小仓位参与
- 稳健型交易者应完全规避
- 持筹者应主动降仓，锁定利润

---

### 特殊情况策略

| 情况 | 策略调整 |
|------|---------|
| **大盘暴跌日(>3%)** | 所有策略仓位减半，禁止追涨 |
| **个股重大利好** | 可适度放宽介入条件，但控制仓位 |
| **涨停板** | 超短线可追，中长线等待开板确认 |
| **跌停板** | 观望，等待企稳信号 |
| **停牌复盘** | 观察首日表现，不急于介入 |

## 六、信号统计格式

```markdown
**综合评分**:
- 趋势强度: __/30分
- 量价配合: __/25分
- 资金流向: __/20分
- 乖离程度: __/15分
- 动量状态: __/10分
- **总分: __/100分**

**背离评估**:
- 量价背离: -__分
- 量能背离: -__分
- 动量背离: -__分
- **背离总分: -__分**

**最终评级**: ⭐_ (__分) - [明确看多/偏多/中性/偏空/明确看空]
```

**重要**：
1. 总分是概率判断，非绝对预测
2. 背离扣分已计入总分，显示为明细便于理解
3. 最终评级基于（总分 - 背离扣分）
